Atlantis Milestone Guardrails (v1.4)

Operational dashboard with guardrails for naming, completion proof, and Discord-ready milestone broadcasts.

1) Iteration Naming Guardrail

Rule: must match v<major>.<minor> and be non-empty (examples: v1.4, v2.0).

2) Completion Gate

Milestone can only be marked done when all checklist items + proof are present.

Gate blocked: complete checklist and add proof.

3) Broadcast Summary (Discord-ready)

Generated automatically when milestone is marked done.

Hosted on GitHub Pages from scoobysteve0/atlantis.
